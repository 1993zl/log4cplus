This describes the release procedure:

 1. Update ChangeLog file with worthy changes.

 2. Run scripts/prepare_dist_from_bzr.sh to prepare tarballs.

 3. Upload tarballs to SourceForge.

 4. Tag revision on branch.

 5. Send announcement to <log4cplus-devel@lists.sourceforge.net>.

 6. Write News entry to SourceForge.

 7. Add news entry to log4cplus.sf.net web page.

 8. Post release information to G+ log4cplus group.
